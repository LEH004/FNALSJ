# **Description:**
    .This is a game made only for pseudocode with no engine
    .This game *ONLY* runs on pseudocode.pro and does not work with any specific syllabus
    .The game only works in "All extra modules" as that provides the window where the game is being rendered on
    .The game is designed to be a challenge to code as a game running without a game engine is hard to produce



# **Controls:**
    Enter: for 'Continue' in menus such as main menu 
    E: Reset SoundSystem
    R: Reset CameraSystem
    S: Play Sound
    1 To 10: Cameras of that Number
    '-' or the minus sign is for camera 11
    C: to exit camera view

# **Installation Guide:**
    1.Dowload the .zip from the github repository https://github.com/LEH004/FNALSJ
    2.To Download click the fropdown menu code and click download zip
    3.After the download is finished extract the folders from the zip 
    4.Drag click all the images and drop them into the pseudocode website where this code is visible
    5.Go into the EP Rooms folder which will have more images
    6.Drop these images into the pseudocode website
    7.Drop the .pseudo file into the pseudocode website too
    8.Open pseudocode settings which is visible as a gear icon at the bottom left
    9.Turn off 'Require Initialisation'
    10.Set 'Max Execution time' to 5000000000 or Higher
    11.Finally its Playable
